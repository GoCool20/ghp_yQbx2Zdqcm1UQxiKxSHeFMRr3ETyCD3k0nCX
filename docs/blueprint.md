# **App Name**: NSE Weekly Gainers

## Core Features:

- Weekly Gainers Display: Loads and displays a paginated table of stocks that have gained more than 2% compared to the same weekday last week, including Symbol, Company Name, Close, Previous Week Close, Percentage Change, Volume, and Comparison Date.
- Dynamic Data Filtering: Allows users to filter the stock data dynamically by a minimum percentage gain (default 2%) and search by stock symbol.
- Interactive Data Sorting: Enables users to sort the displayed stock data table by Highest Gain, Lowest Gain, and Highest Volume with real-time updates.
- Summary Statistics Section: Presents key summary statistics above the data table, such as the total number of stocks currently displayed and the highest percentage gain in the filtered view.
- Responsive Data Table: Ensures the stock data table and filtering controls adapt responsively across different screen sizes, from desktop to mobile.
- Empty State Handling: Displays a clear message, 'No stocks match selected criteria.', if filters result in an empty dataset.
- Informational About Page: Provides a dedicated page detailing how weekly comparisons are calculated and includes a disclaimer about stock market risks.

## Style Guidelines:

- The visual scheme embraces a professional and stable light aesthetic. The primary color is a deep, trustworthy blue (#264D73), chosen to evoke reliability in a financial context. The background is a very light, almost imperceptibly tinted blue-gray (#F3F6F8), offering a clean and expansive canvas. An accent color, a crisp cerulean blue (#4FBAD9), will be used for interactive elements and highlights to ensure clear call-to-actions and important UI states. For financial context, positive percentage changes will be distinctly highlighted in green (#28A745) for immediate recognition, as per user request.
- The font for all text, including headlines and body, is 'Inter' (sans-serif). Its neutral and modern design ensures excellent readability across diverse financial data displays and explanatory text, contributing to the app's clean and professional appearance.
- Minimalist, vector-based icons will be utilized for sorting indicators in table headers (e.g., subtle up/down arrows) and functional elements like information or navigation. Icons will maintain consistency with the clean financial dashboard aesthetic, prioritizing clarity and simplicity over ornamental design.
- The layout is clean, responsive, and inspired by modern financial dashboards. On larger screens, the filtering section will be positioned conveniently alongside the data table. This arrangement fluidly transitions to a single-column layout for mobile devices, placing filters above the table. A fixed header containing the website title ensures constant brand presence, while the content area allows for easy scrolling. Data and summary sections will employ spacious card-like designs with consistent padding for optimal readability and a professional feel.
- Subtle and purposeful animations will enhance user interaction without distraction. These include gentle fade-in effects for content as it loads or updates (e.g., after applying filters), and smooth transitions for table pagination. Interactive elements like buttons and table rows will feature refined hover states for improved usability and feedback.
